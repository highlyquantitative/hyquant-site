Hyquant (as in “Highly Quantitative”) is an AI research project based in Quebec. We focus on measuring how real-world compute behaves using a predictive AI model fully trained by the Hyquant team . Using this information, we can optimize server hardware behavior and improve the way computers use electricity, with the goal of reducing the heavy carbon footprint of servers and hardware waste.
